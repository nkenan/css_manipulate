css_manipulate(Options);

Input

  1. Options

    Options MUST be an array of objects. Every object consists of three properties: 1. selector, 2. key, 3. attributes. Where selector and key hold values of String type the attributes key contains an array of css properties.

    Within a

    • selector

      :String
      An html tag or an id, e.g. 'h1' or '#someId'

    • key

      :String
      A property to set, e.g. 'font-size'

    • value

      :String
      The value for _params.key to set, e.g. '2rem'

    css_manipulate()
